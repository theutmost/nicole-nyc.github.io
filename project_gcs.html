Redesign of Genesis Connected Service’s iOS & Android app

A complete redesign of client's current mobile app, keeping the same framework as Hyundai Blue Link and KIA UVO.

Overview

In Station Digital Media, I worked for designing Genesis Conencted Service to redesign with new look and feel. Genesis Connected is the same as Hyundai Blue Link function which is a connected vehicle system using an embedded telematics to gather vehicle Car Care information it also allows the customer to Remote Start, Stop, Horn, Lock or Unlock.

Genesis Connected Service’s final User Interface: some features are different by its connected vehicle type
Gray/Brown Shades with Gold Accent

Genesis Connected Service has the same framework as Hyundai Blue Link mobile application. However, Genesis Connected Service does not include electric cars. Following up any UI changes made in Blue Link mobile application, I re-skinned design elements including icons, typography, colors, etc.
